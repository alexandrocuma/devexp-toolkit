# Templates

Starting points for creating new agents and skills.

| File | Use For |
|------|---------|
| `agent-template.md` | New agents in `agents/` |
| `skill-template.md` | New skills in `skills/<name>/SKILL.md` |

Both templates are heavily commented. Read the comments — they explain what each field does and how to fill it in effectively.

After filling in a template, run `../install.sh` from the repo root to deploy the new agent or skill.

See `../docs/agent-authoring-guide.md` and `../docs/skill-authoring-guide.md` for comprehensive authoring guidance.
